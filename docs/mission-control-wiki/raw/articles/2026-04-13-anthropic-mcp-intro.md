# Model Context Protocol (MCP) Intro

URL: https://modelcontextprotocol.io/docs/getting-started/intro

Key points:
- MCP is an open-source standard for connecting AI applications to external systems.
- It is the tool/context bridge for letting an agent interact with APIs, files, services, and data.
- MCP is about tool access; it is not the same layer as agent-to-agent communication.
- Hermes should treat MCP as the adapter layer for tools and integrations.
