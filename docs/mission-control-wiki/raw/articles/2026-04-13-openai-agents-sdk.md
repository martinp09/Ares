# OpenAI Agents SDK

URL: https://developers.openai.com/api/docs/guides/agents-sdk

Key points:
- SDK covers agent definitions, orchestration, guardrails, results/state, integrations, observability, and voice agents.
- The architecture is explicitly about running agents, not just prompting models.
- Guardrails and tracing matter if Hermes is going to deploy agents on behalf of the operator.
- This reinforces that the command center needs an execution model plus visibility model.
