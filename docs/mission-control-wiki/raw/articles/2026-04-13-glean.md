# Glean

URL: https://www.glean.com/

Key points:
- Glean positions itself as work AI that works, combining assistant, search, and agents.
- The product surface includes Glean Assistant, Deep Research, Canvas, Companion, Agents, Agent Builder, Agent Governance, and Agent Orchestration.
- Glean emphasizes enterprise graph / personal graph style access to workplace knowledge.
- The company is a strong reference for how to combine search, assistant, agent building, and governance in one platform.
- For Hermes, Glean is a useful model for the control surface around internal knowledge and workforce action.
