# GitHub Agentic Workflows

URL: https://github.github.com/gh-aw/

Key points:
- Repository automation can be written in natural language and run as GitHub Actions.
- GitHub emphasizes guardrails to contain confused or compromised agents.
- Read-only tokens and other security layers are part of the design.
- This is a strong precedent for Hermes: let agents act, but only through controlled surfaces and scoped permissions.
