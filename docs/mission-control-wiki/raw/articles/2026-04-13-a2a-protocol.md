# A2A Protocol

URL: https://a2a-protocol.org/latest/

Key points:
- Open standard for agent interoperability.
- Designed so agents built by different vendors and frameworks can communicate securely.
- Supports communication with remote agents, local agents, and humans.
- Pairs naturally with MCP: use tools through MCP, talk between agents through A2A.
- Relevant for Hermes because Mission Control will increasingly coordinate multiple agents instead of only single-turn prompts.
