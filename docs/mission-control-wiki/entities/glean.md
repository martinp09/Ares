# Glean

Glean is a work AI platform that combines assistant, search, deep research, and agent tooling with governance and orchestration.

Why it matters:
- It shows how to combine retrieval, assistant behavior, and agent building in one platform.
- It treats governance and orchestration as first-class features.
- It is a strong reference for enterprise internal knowledge + action systems.

Hermes takeaways:
- Mission Control should unify search, action, and governance.
- Agent builder surfaces should exist alongside the operator cockpit.
- Internal knowledge and action should be in the same system.
