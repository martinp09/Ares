# Sierra

Sierra is an AI customer-experience platform built around agents that can operate across chat, SMS, WhatsApp, email, voice, and ChatGPT.

Why it matters:
- It demonstrates a channel-native agent product.
- It shows how an agent can be the core service layer across many customer touchpoints.
- It’s a good benchmark for Hermes comms and inbox design.

Hermes takeaways:
- Mission Control should understand channels, not just messages.
- One agent can span many communication surfaces.
- The UI should let the operator see channel context, not hide it.
