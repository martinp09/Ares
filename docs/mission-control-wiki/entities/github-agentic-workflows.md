# GitHub Agentic Workflows

GitHub Agentic Workflows is a strong precedent for Hermes because it shows how to let agents automate repository work while constraining them with guardrails, scoped tokens, and controlled execution surfaces.

Hermes takeaways:
- Agentic systems need permission boundaries.
- A natural-language workflow layer still needs deterministic guardrails.
- The operator should be able to see what the agent is about to do and what it already did.
