# Core Seller Offer Proposal

## Correction / operating rule

Before Ares writes any campaign copy, it must first define the **core offer** being made to the seller.

The copy is downstream of the offer. If the offer is vague, the copy becomes generic investor outreach.

## Proposed core offer

### Offer name

**The Inherited Property Exit Plan**

### One-line offer

A simple way for Harris County families to find out if they can sell an inherited property as-is — without fixing it, cleaning it out, listing it, or having every title/tax question solved before the first conversation.

### Plain-English version

If your family inherited a property and it is becoming more of a project than you wanted, I can review it as-is and tell you whether I can make a practical buyer option work.

You do not need to repair it, clean it out, list it, host showings, or figure out every title/tax issue before we talk.

If keeping it is the plan, no problem — tell me and I will close my notes.

## Hormozi value equation

### Dream outcome

The seller wants relief and clarity:

- know what to do with the inherited property
- avoid turning the house into another family project
- avoid repairs, cleanout, listing prep, open houses, and drawn-out uncertainty
- get a real option without having to become a probate/title/tax expert first

### Increase perceived likelihood

Why they might believe this can work:

- Martin is a local investor/buyer focused on as-is situations.
- The offer is specifically for inherited/probate/property-friction situations, not generic retail listings.
- The process starts with a property review, not a demand that the seller solve every issue first.
- Ares can use property/probate/tax context to prepare a more informed conversation.

### Decrease time delay

Make the first step fast:

- seller can start with a call/text
- no listing appointment required
- no repairs or cleanout before the first conversation
- no need to gather every document before learning whether an as-is option might exist

### Decrease effort / sacrifice

Reduce seller burden:

- no repairs before talking
- no cleanout before talking
- no showings/open houses
- no MLS prep
- no pressure if they are keeping the property
- no need to coordinate the whole family before asking if there is even a buyer option

## Offer stack

What the seller actually gets:

1. **As-is property review** — Martin looks at the property situation as it sits.
2. **Inherited-property friction review** — repairs, cleanout, taxes, title questions, and multiple-heir friction are treated as context, not automatic deal killers.
3. **Simple buyer-option answer** — if it fits, Martin can discuss an as-is path.
4. **No-prep first conversation** — seller does not have to repair, clean, stage, or list first.
5. **Keep/sell clarity** — if selling is not relevant, seller can say so and get removed from follow-up.

## Unique mechanism

### Mechanism name

**As-Is Inherited Property Review**

### Mechanism explanation

Instead of asking the family to first prepare the house for a normal listing, Ares/Martin reviews the inherited-property situation as-is — property condition, probate context, tax/title friction, and seller goal — then decides whether a practical buyer option is worth discussing.

This creates a different category from:

- listing with an agent
- repairing before sale
- generic cash-buyer spam
- trying to solve title/tax questions alone before knowing whether selling even makes sense

## Risk reversal / de-pressure

Use these ideas carefully:

- “If keeping the property is the plan, no problem — just tell me and I’ll close my notes.”
- “If I’m not the right fit, I can still point you toward the next practical step.”
- “No repairs or cleanout before we talk.”
- “No pressure.”

Do **not** say:

- risk-free
- guaranteed close
- guaranteed offer
- guaranteed title solution
- guaranteed tax solution
- we solve probate

## First CTA

Primary CTA:

**“Would it be worth a quick call to see if an as-is option makes sense?”**

Softer CTA:

**“Are you the right person to ask about the property?”**

Direct mail CTA:

**“Call or text me if selling it as-is would help.”**

## Segment-specific offer angles

### HOT

Offer angle:

**A practical as-is option when the inherited property is already becoming a tax/title/repair/cleanup headache.**

Best CTA:

“Would it be worth a quick call this week to see if an as-is option makes sense?”

### WARM

Offer angle:

**A no-pressure way to understand whether selling as-is is even worth considering.**

Best CTA:

“Are you the person handling decisions for the property?”

### COLD

Offer angle:

**Wrong-person-safe inquiry about whether the family would ever consider an as-is sale.**

Best CTA:

“If I have the wrong person, just let me know and I won’t follow up.”

## Offer critique

Strengths:

- It is specific to inherited/probate property friction.
- It reduces time and effort before the first conversation.
- It avoids overclaiming legal/tax/title outcomes.
- It gives sellers an easy exit if they are keeping the property or are the wrong contact.

Weaknesses / needs Martin decision:

- Offer name may need to sound more local/human and less productized.
- Proof points are currently light; we need real examples, testimonials, or “what Martin can actually do” specifics.
- We need decide whether “tax/title questions” should appear in front-end copy or stay as backend personalization.

## Martin approval questions

1. Is **The Inherited Property Exit Plan** the right name, or should it be softer?
2. Should the main promise include **title/tax questions**, or keep that only for HOT leads?
3. Are we comfortable saying “I can still point you toward the next practical step” if it is not a fit?
4. Is the core offer local-Harris only, or should we frame it for all Texas counties from the start?
