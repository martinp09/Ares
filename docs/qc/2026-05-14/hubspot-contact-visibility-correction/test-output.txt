## HubSpot record layout docs check

Public HubSpot knowledge-base pages show that properties shown in record sidebars/property-list cards are a HubSpot UI record-customization setting:

```text
Users can customize which properties are displayed in the left sidebar/property-list cards. On a record, use Actions > Customize properties > Add properties; account-wide default record layout is under Settings > Objects > [object] > Record Customization > Default view, requiring Super Admin / Customize record page layout permissions.
```

The Ares Service Key path can create properties and write property values, but it does not automatically place those properties on Martin's visible HubSpot record card.
