# Test / Smoke Output

## Preview inventory before live apply

Command:

```bash
python - <<'PY'
from app.core.config import Settings
from app.services.hubspot_mirror_service import HubSpotMirrorService
settings = Settings()
service = HubSpotMirrorService(settings=settings)
preview = service.build_customization_preview()
print('hubspot_token_present=' + str(bool(settings.hubspot_access_token)).lower())
print('global_live_gate=' + str(settings.provider_live_sends_enabled).lower())
print('hubspot_live_gate=' + str(settings.hubspot_provider_live_writes_enabled).lower())
print('property_groups=' + str(sum(len(v) for v in preview['payloads']['property_groups'].values())))
print('contact_properties=' + str(len(preview['payloads']['properties']['contacts'])))
print('deal_properties=' + str(len(preview['payloads']['properties']['deals'])))
print('company_properties=' + str(len(preview['payloads']['properties']['companies'])))
print('deal_pipelines=' + str(len(preview['payloads']['pipelines']['deals'])))
print('deal_pipeline_stages=' + str(len(preview['payloads']['pipelines']['deals'][0]['stages'])))
PY
```

```text

```
