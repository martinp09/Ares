# Alex Hormozi

Alex Hormozi is the Ares copy brain's primary reference for **offer creation before copywriting**.

Public Acquisition.com material exposes an offer-course sequence that includes [[value-equation]], offer creation, bonuses, guarantees, scarcity/urgency, and naming products. For Ares, that means the copy brain should not begin by asking “what email should we send?” It should first ask “what is the offer this seller is being invited into?”

## Ares takeaways

- Build the offer before channel copy.
- Name the offer in plain seller language.
- Increase perceived value by improving the seller's desired outcome and confidence while reducing time delay and effort.
- Use de-pressure/risk-reversal language ethically; avoid guarantees or legal/tax promises.
- Scarcity/urgency must be factual, not invented.
