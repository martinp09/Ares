# Pain-First Copy

Pain-first copy starts with the problem the reader already feels, not with the sender's credentials.

## Ares structure

1. Name the situation plainly.
2. Install the [[copy-hinge]] early: what the seller gets for the next few minutes of attention.
3. Create agreement: “that can get messy when...”
4. Contrast the default path that may not fit.
5. Explain the mechanism.
6. Present the offer.
7. Ask for one small next step.

## Probate example pattern

> Inherited property decisions can get messy when repairs, cleanup, taxes, title questions, and multiple family members are all involved. A normal listing is not always the easiest first step. I review Harris County properties as-is and can tell you whether a simple buyer option makes sense before you spend time fixing or cleaning it out.
