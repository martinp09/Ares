# Alen Sultanic Offer Code / Rosetta Stone Source Note

User-provided excerpt summary:

- A working piece of an offer contains the hidden **code** of what the market wants.
- Do not only copy the surface words. Look behind the words:
  - images
  - structure
  - thoughts
  - feelings
  - what the words represent
- Extract the code from the part that is already converting:
  - front end
  - order bump
  - upsell
  - downsell
  - email
  - ad
- Then infuse the code throughout the whole offer, front to back:
  - ads
  - email
  - landing page / VSL
  - bonuses
  - bumps
  - upsells
  - recurring / backend
  - close

## Example pattern from excerpt

Diet offer clues:

- "without spending hours in the kitchen" = time scarcity, need more **without** language.
- "7 smart strategies" = they need numbered steps, process, guides, specific how-to.
- "versatile meals" = life is variable; the solution must adapt to work, kids, dinners, celebrations.
- "almost instant" / "ready in 10 minutes or less" = speed and low energy are repeatedly important.
- "stress-free even when life gets busy" = reduce stress; must work under busy conditions.
- "smart shortcuts to stay on track" = they have fallen off before; offer must help them recover, not demand perfection.
- "drive-thru / gas station" = real-life edge cases matter; add flexibility for on-the-go moments.
- "good enough is great" = remove perfection guilt; give permission for imperfect compliance.

## Core rule

When one part works, ask:

> What is the code underneath this winning asset, and how do we infuse that code everywhere else?

If the offer has no working piece yet, look outside the offer at what is crushing it in the market, extract its code, test it, then infuse what picks up.

## Ares translation

For inherited-property sellers, Ares should look for repeated language that reveals:

- what they want done without doing first
- what they are too busy, overwhelmed, or emotionally tired to handle
- where they need steps, clarity, or a process
- where they need a practical shortcut instead of a perfect process
- which edge cases make the normal path feel impossible
- which working copy element should be propagated into the full funnel
